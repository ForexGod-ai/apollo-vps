📚 SECTION 1: REVERSAL vs CONTINUATION

🔄 Reversal Strategy (Trend Change at Extremes)

Definition:
Trade-ul se deschide după o schimbare majoră de trend pe timeframe superior.

Flow Clasic:
1️⃣ Daily CHoCH (Change of Character) → Trend pe Daily se inversează
2️⃣ 4H CHoCH confirmation → 4H confirmă reversalul
3️⃣ 1H CHoCH/BOS → Entry trigger pe 1H în direcția noului trend
4️⃣ Fibonacci 50% → Calculat pe Daily/4H CHoCH range (macro-structure)

Caracteristici:
✅ Fibonacci calculat pe swing-ul MAJOR (Daily sau 4H)
✅ Stop Loss mai larg (peste/sub Daily swing)
✅ Risk:Reward mai mare (10-30:1)
✅ Necesită dual timeframe confirmation (Daily + 4H + 1H alignment)
✅ Mai multe șanse de pullback la Fibo 50% (trend se stabilizează)

➡️ Continuation Strategy (Pullback in Existing Trend)

Definition:
Trade-ul se deschide pe un pullback într-un trend deja existent.

Flow Clasic:
1️⃣ Daily Trend → Trend clar pe Daily (bullish sau bearish)
2️⃣ 4H Pullback → Prețul face pullback în FVG zone
3️⃣ 1H BOS (Break of Structure) → 1H confirmă reluarea trend-ului
4️⃣ Fibonacci 50% → Calculat pe 1H micro-swing (5-10 candles)

Caracteristici:
✅ Fibonacci calculat pe swing-ul MINOR (1H 5-candle lookback)
✅ Stop Loss mai strâns (sub/peste 1H swing)
✅ Risk:Reward mai mic (3-10:1)
✅ Nu necesită dual CHoCH (doar BOS pe 1H suficient)
✅ Momentum entry posibilă după 6H (fără pullback exact la Fibo)

Aspect REVERSAL CONTINUATION

Dual Timeframe Check ✅ Required (4H CHoCH active) ❌ Not required
Momentum Entry ❌ No (strict pullback) ✅ Yes (after 6H)
Stop Loss Above/below Daily swing Above/below 1H swing

    Verify 4H CHoCH is active when 1H signals entry.
    """

            # Store 4H CHoCH for future reference

            return True
        else:
            logger.warning("4H CHoCH not found - INVALID setup")
            return False
    
    # Check if 4H CHoCH expired (older than 48H)

    if hours_elapsed > 48:
        logger.warning("4H CHoCH expired (>48H old)")
        return False

    latest_choch_4h = detect_choch_and_bos(df_4h)[0][-1]
    
    if latest_choch_4h.direction != setup['direction']:
        logger.warning("4H trend REVERSED - setup invalidated")
        return False
    
    return True  # ✅ DUAL CONFIRMATION VALID
