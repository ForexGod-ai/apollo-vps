Glitch in Matrix — The 3-Layer Strategy
D1 paints the Film → 4H is the Gate → 1H is the Sniper Trigger
🧠
Philosophy: Generals (D1 + 4H) give the attack order — 1H only executes. The Daily candle establishes the "Film" (overall narrative). No trade happens without all three layers aligned. Bot stays completely silent until all 3 conditions are met.

Layer 1 — D1
The Film — Daily Bias Lock
The Daily chart tells us the big story. Is the market in a Reversal pattern (trend change expected) or a Continuity pattern (trend continuation)? This determines the entire trade direction. Everything else MUST align with this.
🔄
REVERSAL
D1 CHoCH detected (HH+HL broken = bearish, LL+LH broken = bullish). Structure flip. Enter on retest of FVG/OB.
➡️
CONTINUITY
D1 BOS detected (trend extension). Price in Premium/Discount zone (50% EQ). Follow institutional direction.
🎯
D1 FVG / POI
The daily Fair Value Gap is the Target Zone where price must return. All lower TF entries must be inside this zone.
⚖️
Premium / Discount
50% equilibrium of the D1 macro leg. BEARISH entry only in Premium (>50%). BULLISH entry only in Discount (<50%).
🔒 LOCK: Strategy type (REVERSAL / CONTINUITY) is saved as strategy_type in setup and never changes for that trade.

Layer 2 — 4H
The Gate — 4H Structure Sync
The 4H timeframe is the confirmation gate. Bot stays completely silent (zero HTTP calls, zero execution) until a 4H CHoCH with BODY CLOSURE confirms the D1 bias direction. A wick through a level is NOT enough — only a candle BODY closing on the other side counts.
🔒
4H Body Closure
CHoCH candle must have its body (max(open,close)) break the swing level. Wicks are rejected as noise. This is the critical V10.4 rule.
⏱️
Max Age 12 Candles
The 4H CHoCH must be recent — no older than 12 × 4H = 48 hours. Stale structure breaks are ignored.
📐
Direction Match
If D1 = REVERSAL BEARISH → 4H CHoCH must be BEARISH too. Mismatched direction = no trade. Period.
🎯
4H Sync FVG
After 4H CHoCH is confirmed, system detects the FVG generated by that confirmation move. This becomes the primary entry zone for 1H.
⛔ GATE: If no valid 4H CHoCH matching D1 bias → bot returns "KEEP_MONITORING" and stays completely silent. Tagged as D1_REVERSAL_4H_SYNC or D1_CONTINUITY_4H_SYNC.

🔄 CHoCH — Change of Character

CHoCH is the primary signal of a trend reversal. It requires strict validation to avoid false signals.

⚠️
Strict Body Closure Rule: CHoCH is only valid when a candle body (max(open,close)) breaks the swing level. Wicks/tails through the level are ignored as noise.
🔍
Dual Pattern Requirement: For BEARISH CHoCH — price must have formed both Higher Highs (HH) AND Higher Lows (HL) before the break. Single pattern → BOS, not CHoCH. This prevents false reversals.
# BEARISH CHoCH validation: if hh_pattern and hl_pattern: # BOTH required # AND body closure check: body_low = min(open[j], close[j]) if body_low < swing.price: # body broke below CHoCH(direction='bearish', ...) # Whipsaw protection: if chochs and (j - chochs[-1].index) < 10: continue # Min 10 candles between CHoCH
📈 BOS — Break of Structure

BOS signals trend continuation. Price breaks a swing in the same direction as the existing trend — confirming continuation.

SignalPattern RequiredMeaning
BULLISH CHoCHLL + LH pattern (downtrend) + body breaks swing highReversal from bearish → bullish
BEARISH CHoCHHH + HL pattern (uptrend) + body breaks swing lowReversal from bullish → bearish
BULLISH BOSAlready bullish trend + body breaks previous swing highContinuation of uptrend
BEARISH BOSAlready bearish trend + body breaks previous swing lowContinuation of downtrend
Key difference: CHoCH requires OPPOSING structure to break. BOS requires SAME-direction break. System checks both in detect_choch_and_bos() and tags each correctly.
🎯 Fair Value Gap (FVG) — Imbalance Zone

An FVG is a 3-candle price imbalance where the market moved too fast, leaving an "unfilled" zone. Institutional traders use these zones to enter as price returns to fill the gap.

🟢 BULLISH FVG (Demand Zone)
# 3-candle gap detection: # body_high[i-1] < body_low[i+1] FVG( direction='bullish', top = body_low[i+1], # Upper edge bottom = body_high[i-1], # Lower edge # Minimum gap size: 0.05% of price ) # MITIGATED when: body_low <= fvg.bottom # (body must close through, not wick)
🔴 BEARISH FVG (Supply Zone)
# 3-candle gap detection: # body_low[i-1] > body_high[i+1] FVG( direction='bearish', top = body_low[i-1], # Upper edge bottom = body_high[i+1], # Lower edge # Minimum gap size: 0.05% of price ) # MITIGATED when: body_high >= fvg.top # (body must close through, not wick)
🔑
Orderflow Alignment (V8.1): FVG search is strictly directional. If CHoCH/BOS is BEARISH → system ONLY searches for BEARISH FVGs. Opposite-direction FVGs are completely ignored. If no classic 3-candle FVG found → Method 2 creates an imbalance zone from swing high/low to the impulse. Proximity filter: only last 30 candles after CHoCH are searched.
⚖️ Premium / Discount Zone Validation (V8.4 Binary Logic)
💡
V8.4 replaced complex percentage calculations with pure binary price comparison. The equilibrium (50% level) is simply (macro_high + macro_low) / 2.0. Above = Premium, Below = Discount.
🔴 BEARISH SHORT — Premium Zone
macro_high ━━━━━━━━ 100%
│ PREMIUM ZONE
equilibrium ━━━━━━━ 50%
│ DISCOUNT ZONE
macro_low ━━━━━━━━ 0%
✅ VALID if: fvg.top ≥ equilibrium
❌ REJECTED if: FVG is entirely below 50% level
🟢 BULLISH LONG — Discount Zone
REVERSAL Logic: Equilibrium measured from PRE-CHoCH macro leg (old trend's extreme to CHoCH break price). Deep retracement (48–52%) into old range = institutional reversal zone.

CONTINUITY Logic: Equilibrium measured from POST-CHoCH impulse leg (current trend's pullback zone). Shallower retracement (38–62%) acceptable in strong trends.
✅ VALID if: fvg.bottom ≤ equilibrium
❌ REJECTED if: FVG is entirely above 50% level
📦 Order Blocks (V3.5)

An Order Block is the last opposing candle before an institutional impulse. When combined with an unfilled FVG nearby, it creates a 10/10 setup.

🔍
Detection
Bullish CHoCH → find last BEARISH candle before impulse. Bearish CHoCH → find last BULLISH candle. Lookback = 10 candles.
📏
Zone Definition
Bullish OB: Body_high to Wick_low. Bearish OB: Wick_high to Body_low. Represents full range of institutional accumulation/distribution candle.
⭐
Scoring (1-10)
Base score 5. +5 if unfilled FVG within 2× OB size nearby → score 10 (PERFECT). +1 if impulse >1% strength. OB + unfilled FVG = "Setup 10/10".
📊 Swing Detection (Body-Based, ATR-Filtered)
✅
All swing detection uses candle bodies (min/max of open and close), NOT wicks/tails. This prevents liquidity sweep spikes from being counted as real swing levels.

Swing Low Algorithm
body_lows = min(df['open'], df['close']) # swing_lookback = 5 (default) for i in range(lookback, n - lookback): current = body_lows[i] left = all(current < body_lows[i-j] for j in range(1, lookback+1)) right = all(current < body_lows[i+j] for j in range(1, lookback+1)) if left and right: swing_lows.append(i)
ATR Filter (V8.2)

ATR (14-period) is calculated to measure volatility. Only swings that represent moves of at least 1.2 × ATR are considered significant. This prevents micro-structure noise from generating false CHoCH signals on range-bound days.

ATR Multiplier: 1.5 (relaxed from earlier versions to capture more valid swings on less volatile pairs like EURCHF).

SMC Glossary
Smart Money Concept definitions as implemented in this system
CHoCH — Change of Character
The first signal of a trend reversal. Price breaks a swing level that requires BOTH patterns to be present (HH+HL for bearish CHoCH, LL+LH for bullish). The breaking candle's BODY must close beyond the level — wicks are rejected.
BOS — Break of Structure
Trend continuation signal. Price breaks a swing level in the SAME direction as the existing trend. No dual-pattern requirement — just the most recent swing high/low in trend direction. Also requires body closure.
FVG — Fair Value Gap
A 3-candle imbalance where the middle candle's movement left a gap: body_high[i-1] < body_low[i+1] (bullish) or body_low[i-1] > body_high[i+1] (bearish). Minimum size: 0.05% of price. FVG is "mitigated" only when a body (not wick) closes through it.
Order Block (OB)
Last opposing candle before a strong institutional impulse. For bullish moves: last bearish candle. For bearish moves: last bullish candle. Zone = body_high to wick_low (bullish OB) or wick_high to body_low (bearish OB). Score 10/10 if unfilled FVG nearby.
Premium Zone
Price above the 50% equilibrium of the macro leg. Considered "expensive" — ideal for SHORT entries. Equilibrium = (macro_high + macro_low) / 2. No ceiling — the higher above 50%, the stronger the Premium.
Discount Zone
Price below the 50% equilibrium of the macro leg. Considered "cheap" — ideal for LONG entries. No floor — the lower below 50%, the stronger the Discount.
Equilibrium (50% Level)
Simple midpoint: (macro_high + macro_low) / 2. For REVERSALS: uses pre-CHoCH macro leg (old trend's extreme to break price). For CONTINUITY: uses post-CHoCH impulse leg (last swing to BOS break price).
Liquidity Sweep
Smart Money manipulation tactic. Equal highs/lows are identified (within 5 pips tolerance). A sweep = wick THROUGH the equal level BUT close back INSIDE. SSL sweep (fake breakdown) precedes bullish CHoCH. BSL sweep (fake breakout) precedes bearish CHoCH. Adds +20 confidence boost.
Fibonacci 50% Entry
Entry target = 50% of the 1H (or 4H/D1 for REVERSAL) swing that created the CHoCH/BOS. Price must touch this level ± tolerance (10 pips standard, 1% for crypto). Entry executed at exact Fibo 50% price, not the wick touch level.
4H Structure Lock
V10.4 gate: D1 bias is set. Now the 4H chart must confirm with a CHoCH in the SAME direction, with body closure, within the last 48 hours (12 × 4H candles). Until this lock is achieved, the bot stays completely silent for that setup.
4H Sync FVG
The FVG generated specifically by the 4H confirmation move (not the D1 FVG). This is the preferred entry zone over the D1 FVG because it's more precise. Only detected after 4H Structure Lock is achieved.
Deep Sleep
System state when daily loss limit (-10%) is hit. ALL scanning stops. Zero HTTP calls, zero CPU usage. Persisted to data/deep_sleep_state.json. Survives process restarts. Wakes at 00:05 UTC next day when daily state resets.
Reversal Strategy
D1 CHoCH detected — the market is changing its primary trend direction. Enter on retest of D1 FVG/OB in the new direction. High R:R setups because catching a trend change early. Requires strictest validation.
Continuity Strategy
D1 BOS detected — the market is extending its existing trend. Enter on pullback to D1 FVG in trend direction. More frequent setups. Still requires 4H + 1H alignment before execution.
Signal Cache / Anti-Spam
V3.8 system. Telegram debouncer prevents identical messages from being sent more than once per 4 hours per reason category. Rejection cooldown: 14,400 seconds (4H). Prevents notification flooding on repeated rejections.
SpatioTemporal Analyzer
Builds narrative (ready_to_trade / monitor_closely / wait / avoid) for each pair by analyzing price action across D1 + 4H + 1H simultaneously. Used by realtime_monitor.py to determine which pairs to deeply scan.
