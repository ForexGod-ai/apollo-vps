## 🔧 TROUBLESHOOTING

### Problem: No OB Detected

**Check:**
```bash
# Verify CHoCH detection
grep "CHoCH detected" daily_scanner.log | tail -10

# Verify FVG detection
grep "FVG detected" daily_scanner.log | tail -10
```

**Solution:**
- OB only detected if CHoCH present
- Not all setups have valid OB (normal behavior)

### Problem: Slow Scans (>60s)

**Check:**
```bash
# Time a scan
time .venv/bin/python daily_scanner.py

# Check data volume
grep "lookback_candles" pairs_config.json
```

**Expected:**
- V3.5: ~40s for 15 pairs (1.67x slower than V3.4)
- If >60s: Check internet connection or API rate limits

### Problem: Telegram Not Sending

**Check:**
```bash
# Test notification
.venv/bin/python -c "
from notification_manager import NotificationManager
nm = NotificationManager()
nm.send_telegram('Test V3.5 Order Blocks ✅')
"
```

**Solution:**
```bash
# Regenerate Telegram token if needed
# See FIX_TELEGRAM_NOTIFICATIONS.md
```

### Problem: OB Score Always 5/10

**Check:**
```bash
# Check FVG correlation logs
grep -A5 "FVG Correlation:" daily_scanner.log | tail -20
```

**Cause:**
- No FVG detected (base score = 5/10)
- FVG too far from OB (distance > 2x OB size)

**Solution:**
- This is normal for some setups
- Focus on 8/10+ setups for high probability

## ✅ HEALTH CHECK SCRIPT

Save this as `health_check.sh`:

```bash
#!/bin/bash

echo "🏥 V3.5 SYSTEM HEALTH CHECK"
echo "=========================="
echo

# 1. Check processes
echo "📍 Process Status:"
SETUP_PID=$(pgrep -f setup_executor_monitor)
POSITION_PID=$(pgrep -f position_monitor)
TRADE_PID=$(pgrep -f trade_monitor)

[ -n "$SETUP_PID" ] && echo "✅ Setup Monitor: Running (PID $SETUP_PID)" || echo "❌ Setup Monitor: Stopped"
[ -n "$POSITION_PID" ] && echo "✅ Position Monitor: Running (PID $POSITION_PID)" || echo "❌ Position Monitor: Stopped"
[ -n "$TRADE_PID" ] && echo "✅ Trade Monitor: Running (PID $TRADE_PID)" || echo "❌ Trade Monitor: Stopped"
echo

# 2. Check lookback config
echo "⚙️  Lookback Configuration:"
grep -A3 '"lookback_candles"' pairs_config.json | grep -E "daily|h4|h1"
echo

# 3. Recent OB detections
echo "📦 Order Block Stats (Today):"
TODAY=$(date +%Y-%m-%d)
OB_COUNT=$(grep "ORDER BLOCK DETECTED" daily_scanner.log | grep "$TODAY" | wc -l | tr -d ' ')
PERFECT_COUNT=$(grep -A3 "ORDER BLOCK DETECTED" daily_scanner.log | grep "OB Score: 10/10" | grep "$TODAY" | wc -l | tr -d ' ')
echo "Total OB Detected: $OB_COUNT"
echo "Perfect 10/10 Setups: $PERFECT_COUNT"
echo

# 4. Active positions
echo "💼 Active Positions:"
if [ -f active_positions.json ]; then
    POS_COUNT=$(cat active_positions.json | grep -o '"positionId"' | wc -l | tr -d ' ')
    echo "Open Trades: $POS_COUNT"
else
    echo "No active positions file found"
fi
echo

# 5. Last scan time
echo "⏱️  Last Scan:"
if [ -f daily_scanner.log ]; then
    tail -1 daily_scanner.log
else
    echo "No scan log found"
fi
echo

echo "✅ Health check complete!"
```

Make executable: `chmod +x health_check.sh`  
Run: `./health_check.sh`
